# Daily Routine & Journey Tracker

A clean, privacy-focused web application that helps you track daily journeys and routines. All data is stored locally in your browser - nothing is sent to any server.

## Features
### Journey Tracking
Log your daily trips with details like mode of transport, distance, time, and notes
### Routine Management 
Create and track daily routine items with simple checkboxes
### Dashboard Overview
See today's journey statistics and routine progress at a glance
### History & Filtering 
Review past journeys with date and transport mode filters
### Data Management
Export your data for backup or import it on another device
### Dark Theme:
Easy-on-the-eyes interface with a modern gradient background

## Technology
Pure HTML, CSS, and JavaScript (no frameworks)
Local storage for data persistence
Responsive design that works on mobile and desktop
No external dependencies or tracking

## Getting Started
Simply open index.html in your browser - no installation or setup required. Your data stays securely in your browser's local storage.
Perfect for commuters, travelers, or anyone wanting to track their daily movements and habits while maintaining complete privacy.
